1. Get the application status by running these commands:
   kubectl get jobs -n {{ include "kube-certs-manager.namespace" . }}
   kubectl get pods -n {{ include "kube-certs-manager.namespace" . }}

2. View the certificate scanner logs:
   kubectl logs job/{{ .Values.job.name }} -n {{ include "kube-certs-manager.namespace" . }} -c cert-scanner

3. View the Slack notifier logs:
   kubectl logs job/{{ .Values.job.name }} -n {{ include "kube-certs-manager.namespace" . }} -c slack-notifier

4. To update the Slack token, create a secret with your token:
   kubectl create secret generic {{ include "kube-certs-manager.secretName" . }} \
     --from-literal=slack-bot-token="xoxb-your-slack-token-here" \
     --namespace={{ include "kube-certs-manager.namespace" . }}

5. To clean up the deployment:
   helm uninstall {{ .Release.Name }} -n {{ include "kube-certs-manager.namespace" . }}
